Sistema de Fazenda

Domine a arte da agricultura medieval: plante, cuide e colha para sustentar seu personagem.

Começando Bancada Recursos Sementes T1 Estações Pecuária

Como Começar sua Fazenda

1

Encontre uma Mesa de Fazenda

A Mesa de Fazenda é a bancada onde você crafta todas as ferramentas de agricultura. Procure uma no mundo ou construa a sua.

2

Craft uma Pá de Plantio e um Balde

Você precisa de uma Pá de Plantio para criar pontos de terra e um Balde de Água para regar suas plantas.

3

Cave a terra e plante suas sementes

Use a pá para criar um Ponto de Terra, depois use a semente no inventário e selecione "Plantar" olhando para a terra.

4

Cuide da planta diariamente

A cada dia, regue com o Balde de Água e aplique Fertilizante para manter a planta saudável. Se ficar sem água ou fertilizante, ela morre!

5

Colha quando estiver pronta

Quando a planta mostrar "Pronta para colheita", pressione E nela para colher seus frutos.

Mesa de Fazenda - Receitas

Todas as ferramentas de fazenda são craftadas na Mesa de Fazenda. Aqui estão as receitas essenciais:

Nível 0

Armazena água para regar plantas. Capacidade: 100%

→ 3x Madeira

Nível 0

→ 3x Lasca de Pedra + 3x Lasca de Madeira

Pó de Osso

Nível 0

Fertilizante fraco. Adiciona +30 fertilização.

→ 1x Osso

Fertilizante

Nível 3

Fertilizante normal. Adiciona +60 fertilização.

→ 1x Osso + 1x Linho

Composteira

Nível 5

Baú especial que converte ingredientes culinários em Fertilizante Avançado (90 fert.) ao passar 1 dia.

→ 4x Madeira + 2x Lingote de Cobre

Água e Fertilizante

Sistema de Água

Como encher o balde:

  • • Entre em água profunda (rio, lago, mar)
  • • Olhe para um Poço de Fazenda
  • • Use um Cache de Água

Cada enchimento: +25% no balde

Regar planta: -25% do balde

Planta recebe: +25 hidratação

Tipos de Fertilizante

Pó de Osso +30
Fertilizante +60
Fertilizante Avançado = 90

* O Fertilizante Avançado define o valor em 90, não soma.

Sistema de Pecuária

Além de plantar, você pode criar animais a partir de ovos obtidos de monstros! Cada criatura tem 10% de chance de dropar um ovo ao morrer.

Como Funciona

1. Mate um

Monstro

2. Colete o

Ovo (10% chance)

3. Coloque no

Ninho ou Pasto

4. Cuide

Água + Comida

5. Colha

Recursos!

Ninhos

Para animais pequenos (aranhas, ratos, sapos, galinhas, etc).

Benefícios:

  • • Produz ovos extras enquanto o animal cresce
  • • Chance de drop diário (40% base)
  • • Colheita final: 2-6 ovos
Ninho Pequeno Ninho Médio

Pastos (Currais)

Para animais grandes (veados, ursos, ovelhas, etc).

Benefícios:

  • • Abate para carne e materiais
  • • Maior quantidade de recursos
  • • Drops: carne, couro, ossos, etc.
Pasto Pequeno Pasto Médio Pasto Reforçado

Cuidados com o Animal

Água

Use o Balde de Água para hidratar. Cada animal consome água diariamente.

• Começa com 25 de água

Comida

Alimente com itens de comida. Animais famintos param de crescer!

• Começa com 25 de comida

Se água ou comida chegar a 0: O crescimento para até você alimentar/hidratar novamente!
